KBAutoFieldBase Class Reference

Inherits from NSObject
Conforms to KBAutoField
Declared in KBAutoField.h

Overview

Base class for predefined autofields, such as KBAutoIntegerField, KBAutoObjectField, KBAutoStringField and KBAutoTimestampField. Available only in non-ARC environment.
