KBAPIConnection Class Reference

Inherits from NSObject
Declared in KBAPIConnection.h

Overview

This is the most important class in the library. To use it you must first create KBAPIRequest, then you should create KBAPIConnection with this request and set the delegate to receive API events such as errors, JSON, XML or autoconstructed objects receiving.
